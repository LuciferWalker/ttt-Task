## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/LuciferWalker/ttt-Task.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Run
   ```sh
   npm run dev
   ```
